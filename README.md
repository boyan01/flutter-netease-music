## How to start

  1. 安装 [Flutter](https://flutter.io/docs/get-started/install) （**注意安装最新的stable版本**）
  2. 在命令行输入下面命令以 profile 模式运行（更快的运行效率）
 ```bash
 flutter run --profile
 ```

## 基本组件依赖

* 页面加载：[**loader**](https://github.com/boyan01/loader)
* Toast及应用内通知： [**overlay_support**](https://github.com/boyan01/overlay_support)
